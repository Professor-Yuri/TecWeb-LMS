Objetivo

Capacitar o aluno a implementar algoritmos para a solução de problemas computacionais em uma linguagem de programação orientada a objetos.

Carga Horária

50 horas de carga horária, divididas em 3 horas de aula semanais.

Conteúdo Programático - Aulas

  1. Classes e objetos (3h)
    1. Instanciação
  2. Atributos (2h)
    1. Públicos
    2. Privados
    3. Finais
  3. Métodos (3h)
    1. Públicos
    2. Privados
    3. Estáticos
  4. Modelando objetos do mundo real (5h)
  5. Relacionamento entre objetos (7h)
    1. Mensagens
    2. Composição
    3. Herança
    4. Classes abstratas
  6. Exceções (2h)
  7. UML (2h)
    1. Um exemplo de modelagem com UML
  8. Interface com o usuário (5h)
  9. Padrões de projeto (14h)
    1. Expert
    2. Interface
    3. Polimorfismo
    4. MVC
  10. Testes de Software (7h)
    1. Testes de unidade
    2. Acceptance Test Driven Development – ATDD
